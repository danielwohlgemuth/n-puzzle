## Instrucciones
En caso de que la libreria de interfaz gráfica Tkinter para Python 3 no esté instalada se puede hacerlo con

    sudo apt install python3-tk

Aparte de eso el programa es ejecutable con el comando

    python3 n-puzzle.py
